# About Toruneko Tool
- ドラゴンクエストキャラクターズ トルネコの大冒険3 不思議のダンジョン の攻略ツールです
- 毎回変化する未識別アイテムの識別を手助けします
- 金額によるアイテムの絞り込み機能を実装しました
- まだすべての未識別アイテム名を網羅していませんのでご注意ください
- C#.NETによるデスクトップアプリケーションとして作成しました
